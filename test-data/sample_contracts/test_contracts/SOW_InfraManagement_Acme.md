## 1. PROJECT OVERVIEW

### 1.1 Project Name
Infrastructure Modernization and Cloud Migration Program

### 1.2 Project Description
Provider shall deliver comprehensive IT infrastructure management services including data center operations, cloud migration, network management, and 24/7 support services for Client's North American operations.

### 1.3 Business Objectives
- Achieve 99.9% system availability
- Reduce infrastructure costs by 25% within 12 months
- Migrate 80% of workloads to cloud by project end
- Improve incident response times by 40%

---

## 2. SCOPE OF SERVICES

### 2.1 In-Scope Services

#### 2.1.1 Data Center Operations
- Server management (250 physical servers, 1,200 virtual machines)
- Storage management (500 TB total capacity)
- Backup and disaster recovery operations
- Environmental monitoring

#### 2.1.2 Cloud Services
- AWS infrastructure management
- Azure Active Directory administration
- Cloud cost optimization
- Multi-cloud orchestration

#### 2.1.3 Network Services
- WAN management (45 locations)
- LAN management (12,000 endpoints)
- Firewall and security appliance management
- VPN administration

#### 2.1.4 Support Services
- 24/7/365 helpdesk support
- Tier 1, 2, and 3 technical support
- Incident and problem management
- Change management

### 2.2 Out-of-Scope Services
- Application development
- End-user device procurement
- Physical security services
- Telecommunications carrier contracts
