## Platform Capabilities

### 1. Contract Intelligence (AI-Native)

**Automated Extraction**
- Upload PDF/DOCX contracts (single, batch, or ZIP)
- AI extracts metadata: parties, dates, values, jurisdiction
- Classify into 6 contract types: MSA, NDA, SOW, SLA, Amendment, Vendor
- Calculate risk scores across 10 risk categories

**Clause-Level Understanding**
- Identify and classify 17+ clause types
- Section-level granularity with page references
- High-risk clause flagging with explanations
- Clause comparison against standard templates

**Natural Language Q&A**
- Ask questions in plain English
- RAG-powered answers with source citations
- Cross-contract search and comparison
- "What are my termination rights with Acme Corp?"

### 2. Obligation & Compliance Management

**Obligation Tracking**

- RAG status (Red/Amber/Green) based on deadlines

**Compliance Dashboard**
- Real-time compliance rate tracking
- Overdue obligation alerts
- Category breakdown (Payment, Delivery, Reporting, etc.)

## Quick Start

### Prerequisites

- Docker and Docker Compose
- OpenAI API key
- Node.js 18+

### Installation

```bash
# Clone the repository
git clone https://github.com/your-org/evaluetor.git
cd evaluetor

# Configure environment
cp .env.example .env
# Add your OPENAI_API_KEY to .env

# Start infrastructure
docker-compose up -d

# Set up backend
cd backend
uv sync
uv run alembic upgrade head
uv run uvicorn app.main:app --reload --port 8000

# Set up frontend (new terminal)
cd frontend
npm install
npm run dev
```
