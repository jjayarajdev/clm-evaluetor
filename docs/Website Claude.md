# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

Evaluetor is a marketing/product website for a Contract Lifecycle Management (CLM) cloud service by Senority B.V. (Rotterdam, Netherlands). The site was generated using **Incomedia WebSite X5 Evolution 11.0.6.27** and is a static website with PHP backend components.

## Development

There is no build system, package manager, or test framework. The site is static HTML/CSS/JS with PHP for server-side features.

**Local preview:** Open any `.html` file directly in a browser. PHP features (admin panel, contact forms, search) require a PHP-capable web server (e.g., `php -S localhost:8000`).

**Deployment target:** `http://www.evaluetor.com` (see `sitemap.xml` and `res/x5settings.php`).

## Architecture

### Frontend Engine
All pages load jQuery (`res/jquery.js`) and the WebSite X5 engine (`res/x5engine.js`), which deferred-loads additional scripts: `x5engine.deferrable.js`, `l10n.js`, `x5cartengine.js`, `x5settings.js`. The engine uses a boot queue (`x5engine.boot.push(...)`) to initialize components after DOM ready.

### Page Structure
- **Public pages** (root): `index.html`, `relationship-management.html`, `contract-management.html`, `service-management.html`, `improvement-management.html`, `platform.html`, `clients.html`, `contact.html`
- Each page has a corresponding per-page CSS file in `pcss/`
- Shared layout CSS is in `style/` (reset, style, template, menu, print, IE fixes)
- Navigation menu is configured in `res/x5settings.js` and rendered by the x5engine menu system

### Backend (PHP)
- `admin/` — Admin panel with session-based login (`checkaccess.php`), blog management, guestbook, and a website test page
- `res/x5engine.php` — Core PHP engine (~190KB, handles sessions, email, access control, search)
- `res/x5settings.php` — Site configuration (URL, email templates, mailer settings)
- `res/imemail.inc.php` + `class.phpmailer.php` + `class.smtp.php` — PHPMailer-based email system
- `captcha/` — Simple image-based CAPTCHA system
- `imsearch.php` — Site search functionality

### Key Conventions
- Language: Dutch (`nl-NL`), content is mixed Dutch/English
- Fixed-width layout at 960px (`meta viewport width=960`)
- All element IDs follow WebSite X5 naming: `imCell_N`, `imTextObject_N`, `imObjectImage_N`, etc.
- CSS classes use `im` prefix (e.g., `imMnMn`, `imGrid`, `imClear`, `imHidden`)
- The `res/` directory contains framework files — avoid modifying these unless necessary as they are generated by WebSite X5
